# SiReSPo - Simplest React Starter Possible

This project is set up using `yarn` with following packages:
- webpack + babel
- webpack-dev-server
- html-webpack-plugin

## Setup
- `git clone git@github.com:David3141/react-starter.git`
- `yarn install`

## Development
- `yarn dev`

## Compiling
- `yarn compile` or `yarn compile-production`
